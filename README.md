## Reminder para mim mesmo

### Requerimento:

- ghcup - Gerenciador de versão do haskell

- cabal - Package manager do haskell

- node.js - Javascript

- npm - Gerenciador de pacotes do node.js

### Fazer build do site:

```bash
cabal clean
cabal update
cabal build
```


### Limpar cache

```bash
rm -rf _cache
```

### fazer build do CSS:

```bash
npm install
npm run build
```
