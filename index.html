Esse site

Esse site foi feito com intenção compartilhar minha jornada no ensino superior, ele foi feito usando hakyll. Feito por Jasper Van der Jeugt, hakyll é um gerador de sites estáticos em uma linguagem de programação funcional chamada Haskell. Também uso TailwindCSS pois eu acho que fazer stylesheets CSS manualmente uma perda de tempo.
